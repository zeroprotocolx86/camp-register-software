# 📖 Інструкція користувача CampRegister

## Швидкий старт

### Встановлення

**Оберіть версію:**

**Варіант А: Інсталятор (рекомендовано)**
1. Завантажте `CampRegister_Setup.exe` (2.2 MB)
2. Запустіть файл
3. Слідуйте інструкціям майстра встановлення
4. Програма з'явиться в меню Пуск

**Варіант Б: Portable (без встановлення)**
1. Завантажте `CampRegister_Portable.exe` (10.59 MB)
2. Помістіть файл в зручне місце
3. Двічі клацніть для запуску

---

## Перший запуск

1. Запустіть програму
2. Введіть **Місце** (назва села/міста)
3. Оберіть **Дату** табору
4. Натисніть **"Почати реєстрацію"**

---

## Реєстрація дитини

1. **Прізвище та ім'я:** Іванов Іван
2. **Вік:** 10
3. Натисніть **Enter** або **"Зареєструвати"**
4. Підтвердіть ім'я на бейджику
5. Натисніть **"Друкувати"** або **"Закрити"**

## Основні функції

### Друк бейджиків
- Розмір: 70×30 мм
- Автоматичне масштабування тексту
- Попередній перегляд

### Експорт в Excel
1. Натисніть **"Excel"**
2. Оберіть місце збереження
3. Файл містить всі дані + статистику

### Мережева робота
1. Перейдіть в **"Мережа"**
2. Оберіть режим:
   - **Авто** - автоматичне визначення
   - **Сервер** - головний ПК з принтером
   - **Клієнт** - допоміжні ПК

### Backup даних
1. Натисніть **"Бекап даних"**
2. Оберіть папку
3. Збережіть JSON файл

**Завжди робіть бекап перед закриттям!**

## Налаштування груп

1. Головна → **"Налаштування"**
2. Змініть вік, назву, кольори груп
3. Натисніть **"Зберегти налаштування"**
