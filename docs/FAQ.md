# ❓ Часті питання (FAQ)

## 📥 Встановлення та запуск

### Яку версію обрати: Portable чи Setup?

**Portable (рекомендовано для початківців):**
- ✅ Не потребує встановлення
- ✅ Можна запускати з флешки
- ✅ Не залишає слідів в системі
- ❌ Розмір 10.6 MB

**Setup (інсталятор):**
- ✅ Інтеграція з Windows
- ✅ Іконка в меню Пуск
- ✅ Менший розмір 2.2 MB
- ❌ Потребує встановлення

### Програма не запускається. Що робити?

1. **Перевірте Windows:** Програма працює тільки на Windows 10/11
2. **Відключіть антивірус** на 1 хвилину і спробуйте знову
3. **Запустіть від імені адміністратора:** Правою кнопкою → "Запустити від імені адміністратора"
4. **Перевірте WebView2:** Має бути вбудований в Windows, але якщо немає - завантажте з [офіційного сайту](https://developer.microsoft.com/microsoft-edge/webview2/)

### Антивірус блокує програму

Це **хибне спрацювання**. Програма на 100% безпечна, написана на Rust + Tauri (сучасна технологія).

**Як додати в виключення:**
1. Windows Defender: Параметри → Оновлення та безпека → Безпека Windows → Захист від вірусів → Керування параметрами → Виключення → Додати виключення
2. Додайте файл `CampRegister_Portable.exe`

## 🖨️ Друк

### Бейджики не друкуються

**Перевірте:**
1. ✅ Принтер підключено та увімкнено
2. ✅ Драйвер принтера встановлено в Windows
3. ✅ Служба "Print Spooler" запущена:
   - Натисніть `Win+R`
   - Введіть: `services.msc`
   - Знайдіть "Print Spooler"
   - Якщо зупинено → Правою кнопкою → "Запустити"

### Бейджики друкуються не того розміру

**Налаштуйте принтер:**
1. Відкрийте діалог друку Windows
2. Натисніть "Налаштування принтера"
3. Оберіть розмір паперу: **70mm × 30mm**
4. Орієнтація: **Альбомна** (горизонтальна)

### Текст не влазить на бейджик

Програма **автоматично підлаштовує** розмір тексту! Але якщо ім'я дуже довге (20+ символів), використовуйте зменшувальне:
- Замість "Олександр-Володимир" → "Саша"
- Замість "Катерина-Марія" → "Катя"

## 💾 Дані

### Де зберігаються дані?

**Локально на вашому комп'ютері:**
```
C:\Users\ВашеІм'я\AppData\Roaming\com.camp.register\camp_register_data\
```

Дані **НЕ відправляються** в інтернет. Все залишається на вашому ПК.

### Дані зникли після перезапуску

**Це неможливо!** Дані зберігаються автоматично після кожної реєстрації.

**Можливі причини:**
1. Ви запустили програму на іншому комп'ютері
2. Ви використали іншого користувача Windows
3. Дані були видалені вручну

**Рішення:** Відновіть з бекапу (якщо робили)

### Як перенести дані на інший комп'ютер?

**Метод 1: Бекап (рекомендовано)**
1. На старому ПК: Натисніть "Бекап даних"
2. Збережіть файл на флешку
3. На новому ПК: Натисніть "Відновити"
4. Оберіть файл бекапу

**Метод 2: Копіювання папки**
1. Скопіюйте всю папку:
   ```
   C:\Users\ВашеІм'я\AppData\Roaming\com.camp.register\
   ```
2. Вставте в те ж місце на новому ПК

## 🌐 Мережа

### Клієнт не може підключитися до сервера

**Перевірте:**
1. ✅ Всі комп'ютери в **одній Wi-Fi мережі**
2. ✅ Сервер **запущено** (зелена галочка)
3. ✅ IP адреса **правильна**
4. ✅ Firewall Windows дозволяє з'єднання:
   - Панель керування → Брандмауер Windows
   - "Додаткові параметри"
   - "Правила для вхідних з'єднань" → "Створити правило"
   - Порт: **33445**, Протокол: **TCP**

### Як дізнатися IP адресу сервера?

**На комп'ютері-сервері:**
1. Відкрийте програму
2. Перейдіть в "🌐 Мережа"
3. Оберіть режим "Сервер"
4. IP адреса показана на екрані (наприклад: `192.168.1.100`)

**Або через командний рядок:**
1. Натисніть `Win+R`
2. Введіть: `cmd`
3. Введіть: `ipconfig`
4. Знайдіть "IPv4 Address" для вашого Wi-Fi адаптера

### Режим "Авто" не знаходить сервер

**Рішення:**
1. Переключіться на ручні режими
2. На головному ПК: оберіть "Сервер" та запустіть
3. На інших ПК: оберіть "Клієнт" та введіть IP адресу сервера

### Чи потрібен інтернет для мережевої роботи?

**НІ!** Потрібна тільки **локальна мережа** (Wi-Fi роутер). Інтернет **НЕ потрібен**.

## 📊 Excel експорт

### Excel каже "файл пошкоджено"

Це **відома помилка Excel**, файл насправді **нормальний**!

**Рішення:**
1. Натисніть **"Так"** в діалозі Excel
2. Excel автоматично відновить файл
3. Файл відкриється нормально

**Альтернатива:**
- Використовуйте **LibreOffice** (безкоштовно)
- Або **Google Sheets** (завантажте файл туди)

### Як відкрити Excel файл на телефоні?

1. Скопіюйте файл на телефон
2. Відкрийте в додатку **Microsoft Excel** (Android/iOS)
3. Або завантажте в **Google Sheets**

## ⚙️ Налаштування

### Як змінити кольори груп?

1. Головна → **"Налаштування"**
2. Прокрутіть до **"Налаштування груп"**
3. Клацніть на колір → оберіть новий
4. Натисніть **"Зберегти налаштування"**

### Як додати нову групу?

1. Головна → **"Налаштування"**
2. Натисніть **"+ Додати групу"**
3. Вкажіть вік, назву та кольори
4. Натисніть **"Зберегти налаштування"**

### Як змінити вікові межі груп?

**Приклад:** Хочу щоб Група I була 7-9 років (замість 6-8)

1. Головна → **"Налаштування"**
2. Змініть "Мін. вік" на 7
3. Змініть "Макс. вік" на 9
4. Натисніть **"Зберегти налаштування"**

## 🔄 Бекап

### Як часто робити бекап?

**Рекомендації:**
- ✅ В кінці кожного робочого дня
- ✅ Перед закриттям програми
- ✅ Після великої кількості реєстрацій (50+ дітей)

### Куди зберігати бекап?

**Рекомендуємо:**
1. 📁 Папка "Документи"
2. 💾 Флешка (для безпеки)
3. ☁️ Google Drive / Dropbox (синхронізація з хмарою)

### Що містить бекап?

Бекап **JSON файл** містить:
- Всі табори (місця та дати)
- Всіх зареєстрованих дітей
- Налаштування груп
- Налаштування друку

**НЕ містить:**
- Мережеві налаштування
- Історію друку

## 🐛 Інші проблеми

### Програма зависла / не відповідає

1. Зачекайте 10-15 секунд
2. Якщо не допомагає: `Ctrl+Alt+Delete` → Диспетчер задач → Завершити задачу
3. Запустіть програму знову
4. Дані **збережені** (програма зберігає після кожної реєстрації)

### Програма працює повільно

**Можливі причини:**
1. Старий комп'ютер (менше 4 GB RAM)
2. Багато програм відкрито одночасно
3. Антивірус сканує програму в реальному часі

**Рішення:** Додайте програму в виключення антивірусу

### Як видалити програму?

**Portable версія:**
- Просто видаліть файл `CampRegister_Portable.exe`

**Setup версія:**
- Панель керування → Програми та компоненти → CampRegister → Видалити

**Дані залишаться** в папці:
```
C:\Users\ВашеІм'я\AppData\Roaming\com.camp.register\
```
Видаліть її вручну якщо потрібно.
